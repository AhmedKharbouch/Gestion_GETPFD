Developer Biography

Application Presentation:

Hello everyone,

I am delighted to present to you this new web application for managing product suppliers and warehouses. This application has been developed using Spring Boot Microservices for the backend, Angular for the frontend, and an R API for the Apriori algorithm, although it has not yet been implemented in the application.

The application utilizes an H2 Database to ensure efficient information management. Additionally, the frontend is secured with Keycloak, providing an added layer of protection.

I would also like to emphasize that the source code of this application is available on GitHub, allowing you to access it and explore the features in detail.

In summary, the main features of this application are as follows:

  • Management of product suppliers and warehouses.
  • Developed using Spring Boot Microservices for the backend.
  • Utilizes Angular for the frontend.
  • Integration of an R API for the Apriori algorithm (not yet implemented in the application).
  • H2 Database for efficient data management.
  • Frontend secured with Keycloak.
  • Source code available on GitHub.

Thank you for your attention. If you have any questions or comments, please feel free to ask.
